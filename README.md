# HR Analytics Dashboard

## Objective
Analyze employee demographics, salaries, performance ratings, and workforce trends.

## Tools
- Power BI
- Excel
- DAX
- Power Query

## KPIs
- Department-wise employee count
- Gender distribution
- Salary analysis
- Top earners
- Performance ratings
- Company growth trends

## Key Insights
- Department X has highest workforce.
- Salary and performance show moderate correlation.
- Country Y has highest average salary.
